GEO GAMBLER
===========

[Rallyteam Project](https://hack.rallyteam.com/#/opportunities/51)

### Stack

* Frontend
  * ES6 Javascript
  * HTML5 Geolocation API
  * CSS Flexbox
* Backend
  * Go 1.7
  * Goa framework
  * In-memory datastore
